Översikt – Lärandemål och innehåll

Kartan hjälper dig hitta rätt modul/verktyg för varje lärandemål.

Kunskap och förståelse

  • Koder (BCD/NBCD, alfanumeriska, Excess-3, Gray) → Modul 01, Kodomvandlare
  • 2- och 10-komplement → Modul 01, Adderare, Mul/Div
  • Datorns uppbyggnad (styrenhet, dataväg, registerfil, ALU) → Modul 03, Datapath-sim
  • Automatisk styrenhet, styrsekvenser → Modul 03, Datapath-sim
  • Lagrade programmets princip → Modul 03, Modul 07
  • Instruktionsuppsättning och adresseringssätt → Modul 07, ISA-sim
  • Undantagshantering → Modul 07
  • Minnesteknologier och minneshierarki → Modul 05
  • Bussprotokoll, multiplex → Modul 06, Buss-sim

Färdigheter och förmåga

  • Basomvandlingar, 2-komplement, mul/div → Basomvandlare, Adderare, Mul/Div
  • Kombinatoriska nät (väljare/heladderare) → Modul 02, Sanningstabell
  • Sekvensnät (minneselement/räknare) → Modul 02
  • ALU och styrsignaler → Modul 04, ALU, Datapath
  • Dataväg: register ↔ ALU ↔ minne → Modul 03, Datapath
  • Assembler/subrutiner, assemblera/disassemblera → Modul 07, ISA-sim
  • Avkodningslogik för primärminne → Modul 05, Decoder
  • Minnesadresserad IO → Modul 06, MMIO

Värderingsförmåga och förhållningssätt

  • HW/SW-samspelet (prestanda/energi/realtid/säkerhet) → Reflektionsfrågor i modulerna 03, 05, 06, 07.
  • Styrsekvenser utifrån instruktioners semantik → Datapath-sim, ISA-sim
  • Konstruera sekvensnät, avkodning, MMIO → M02, M05, M06
